A programming language is a special language that is used by programmers to write software or scripts or another set of commands to be executed on a computer C language: It is a high-level programming language of general use, created in the seventies of the last century in laboratories Bell by Dennis Ritchie It is an ideal language for writing fixed and mobile programs. The Java language is a high-level programming language of general use, in addition to being object oriented, it was created by Sun Microsystems in 1991 AD and is currently subject to Oracle after the latter purchased Sun in 2009. According to a statistic from Oracle, at the present time, there are more than 9 million programmers that use the Java language.
